# Excel Data Wrangling Project

**Overview:**  
This project demonstrates data wrangling and analysis skills in Excel. It uses three mock datasets to practice importing, cleaning, transforming, and analyzing data. The project emphasizes both accuracy and professional presentation.

**Project Goals:**  
- Import and merge multiple datasets  
- Clean and format data for analysis  
- Apply Excel formulas for data manipulation  
- Use pivot tables and charts to analyze trends  
- Summarize insights clearly

**Tools Used:**  
- Microsoft Excel  
- Pivot Tables  
- Formulas (VLOOKUP, XLOOKUP, IF statements, CONCATENATE, TEXT functions)  
- Conditional formatting  
- Charts and graphs

**Project Steps:**  

**Part A – Wrangling:**  
- Import and combine Data_Set_2.csv and Data_Set_3.csv into Data_Set_1.csv  
- Format data as a table with named columns  
- Sort data by ID  
- Add missing columns (UserName, CompanyName, JobTitle) using formulas  
- Create a `FullName` column combining first and last names  
- Clean credit card type names and filter for American Express users  

**Part B – Analyzing Customer Demographics:**  
- Count unique customers by race and gender  
- Summarize total sales by gender and race  
- Create a chart illustrating total sales by gender and race  

**Part C – Analyzing Customer Sales:**  
- Identify top 5 customers by number of purchases  
- Identify top 5 customers by total purchase value  
- Pull in associated company names for top customers  

**Part D – Wrangling:**  
- Extract month, month number, and year from dates  
- Prepare data for further analysis  

**Part E – Analyzing Sales (Pivot Tables):**  
- Analyze total sales by year, month, company, credit card, and state  
- Identify trends, top performers, and summarize findings

**Summary of Skills Demonstrated:**  
- Data cleaning and wrangling  
- Excel formula proficiency  
- Pivot table creation and formatting  
- Chart creation and visualization  
- Data analysis and summarization  
- Professional presentation of results
